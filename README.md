# FinGuard AI – Intelligent Financial Fraud Detection & Banking Assistant

## Project Overview

FinGuard AI is an AI-powered banking application that detects fraudulent transactions and answers banking-related questions using Retrieval-Augmented Generation (RAG).

The project combines Machine Learning, FastAPI, React, MySQL, and LangChain to provide an intelligent banking assistant.

---

## Features

- Fraud Detection using Machine Learning
- Banking Question Answering using RAG
- FastAPI Backend
- React Dashboard
- Banking Knowledge Chatbot
- Transaction History
- Fraud Statistics Dashboard
- MySQL Database Integration

## Installation

### Clone Repository

```bash
git clone https://github.com/vtu27960-sanjay/FinGuard-AI.git
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm start
```
